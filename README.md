# NexaAI - Premium SaaS Landing Page Template

NexaAI is a high-performance, modern landing page built with **React.js** and **SCSS**. Designed specifically for AI startups, cybersecurity firms, and SaaS businesses.

## ✨ Features
- 💎 **Modern UI:** Clean glassmorphism design.
- 📱 **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.
- 🚀 **Waitlist Ready:** Built-in modal for lead collection.
- ⚡ **Performance:** Extremely fast loading with optimized assets.

## 🛠️ Installation
1. Clone or extract the files.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the local development server.
4. Run `npm run build` to create a production-ready build.

## 🎨 Customization
- **Logo:** Replace `/public/nexa-ai.png` with your own logo.
- **Colors:** Update the variables in `src/components/header/Normal/Header.scss`.
